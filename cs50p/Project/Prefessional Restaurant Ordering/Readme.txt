# Professional Restaurant Ordering System - Single Item MVP

A restaurant ordering system built with python.
This project demonstrates beginner level input handling, conditional logic and formatted output.

The program allow user to:
- Select one item from the menu
- Enter quantity
- Automatically calculate subtotal, discount, tax and total prices.
- Display a clean and formatted receipt

Prices are represented in *k*
(Thousands of VND)

---
Features
- Input validation with "try/expect"
- Automatic adding 5% discount for orders over 200k
- Tax applied after discount
- Clean receipt formatting using f-string
- Exit on invalid inputs
